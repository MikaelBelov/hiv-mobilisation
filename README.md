# Replication Package: Military Mobilisation and HIV Transmission in Russia

Replication data and code for "Military mobilisation and HIV transmission in Russia: a difference-in-differences study."

## Quickstart

```bash
python run_all.py
```

This reproduces all results from raw data: PDF extraction, panel construction, main analysis, and figures. Runtime ~2 minutes. Requires Python 3.9+ with `pandas`, `numpy`, `statsmodels`, `scipy`, `matplotlib`, `pdfplumber`, and `openpyxl`.

## Data Sources

### HIV testing data (Form No 4)

Regional HIV testing and detection data from information bulletins nos 34-50, published by:

> Federal Scientific and Methodological Centre for AIDS Prevention and Control (Referens-tsentr po monitoringu za VICh i VICh-assotsiirovannymi infektsiyami), Federal Budgetary Scientific Institution Central Research Institute of Epidemiology, Rospotrebnadzor.

Source: http://www.hivrussia.info/elektronnye-versii-informatsionnyh-byulletenij/

Note: the website is accessible only from Russian IP addresses. PDF bulletins are included in `raw_data/bulletins/`.

### Household bank deposits (Central Bank of Russia)

Monthly household deposit data by region, Table 02.06 "Deposits of individuals" (Vklady (depozity) fizicheskikh lits).

- Portal: https://cbr.ru/statistics/bank_sector/sors/
- Direct file: https://cbr.ru/vfs/statistics/BankSector/Borrowings/02_06_Dep_ind.xlsx

Note: the website is accessible only from Russian IP addresses. The xlsx file is included in `raw_data/cbr/`.

Methodology follows Solanko L. "Where do Russia's mobilized soldiers come from? Evidence from bank deposits." *BOFIT Policy Brief* 2024; 1: 1-12.

### Military fatalities (Mediazona)

Independently verified list of Russian military deaths in Ukraine, attributed to home regions. Source: https://200.zona.media

### Population denominator

Male working-age population (16-61.5 years) from the 2021 Russian Census. Follows the methodology of Bessudnov A. "Ethnic and regional inequalities in Russian military fatalities in Ukraine." *Demographic Research* 2023; 48: 883-98.

## Repository Structure

```
raw_data/
  bulletins/          # PDF bulletins nos 30-50 from Federal AIDS Centre
  cbr/                # Central Bank deposit statistics (02_06_Dep_ind.xlsx)
  census/             # 2021 census male working-age population
  mediazona/          # Confirmed military fatalities by region and year

scripts/
  extract_codes_2009_2017.py   # PDF -> HIV codes for bulletins 34-43
  extract_codes_2018_2021.py   # PDF -> HIV codes for bulletins 44-47
  extract_codes_2022_2024.py   # PDF -> HIV codes for bulletins 48-50
  normalize_regions.py         # Canonical region name harmonisation
  process_cbr_deposits.py      # CBR xlsx -> deposit growth series
  build_full_panel.py          # Assemble all sources into panel
  analysisv2.py                # Main analysis (all results in paper)
  figures.py                   # Generate all figures

output/                        # Generated by run_all.py (not tracked in git)
  hiv_codes_*.csv              # Extracted HIV codes
  deposit_yoy_*.csv            # Processed deposit data
  full_panel.csv               # Final analysis panel
  analysis_output.txt          # All numerical results
  fig*.png, fig*.pdf           # Figures

run_all.py                     # Master script: raw data -> all results
full_panel_codebook.md         # Variable definitions and data documentation
```
